# Docs Overview

このリポジトリの開発関連ドキュメントは `docs` 配下で管理する。

## フォルダ構成
- `docs/requirements/`
  - 要件定義を置く
- `docs/plans/`
  - 方針、決定事項、実施順序、作業メモを置く
- `docs/manuals/`
  - 利用者向けの操作マニュアルを置く
- `docs/tasks/`
  - 開発タスク、進捗、未対応項目を置く

## 運用ルール
- 今後の方針や判断事項は、会話だけで終わらせず Markdown に残す
- 要件の更新は `docs/requirements/` に反映する
- 実装方針や設計判断は `docs/plans/decision_log.md` に追記する
- 開発タスクと進捗は `docs/tasks/colab_migration_tasks.md` で管理する

## 現在の主要ファイル
- 要件定義: `docs/requirements/colab_requirements.md`
- 決定ログ: `docs/plans/decision_log.md`
- 実装設計: `docs/plans/colab_implementation_design.md`
- 次ステップ: `docs/plans/next_steps.md`
- 利用手順: `docs/plans/colab_usage.md`
- 操作マニュアル: `docs/manuals/colab_user_manual.md`
- 検証チェック: `docs/plans/verification_checklist.md`
- タスク管理: `docs/tasks/colab_migration_tasks.md`
